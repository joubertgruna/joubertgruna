## 📱 TESTE RÁPIDO (5 MINUTOS)

### Setup
```bash
1. Abrir DevTools (F12)
2. Cmd+Shift+M (mobile view)
3. iPhone 12 Pro
4. http://localhost:5173/feed
```

### Testes
```
✓ Feed carrega?
✓ Items aparecem?
✓ Imagens visíveis?

→ SWIPE DIREITA
  ✓ Aparece coração verde (♥)?
  ✓ Network tab: POST /api/likes?
  ✓ Item desaparece?

← SWIPE ESQUERDA
  ✓ Aparece X vermelho (✗)?
  ✓ Network tab: DELETE /api/likes?
  ✓ Item desaparece?

TAP RÁPIDO
  ✓ Abre detalhe (/items/{id})?
  ✓ Mostra item grande?

VOLTAR
  ✓ Volta ao feed?
  ✓ Items que curtiu não aparecem?
```

## 📊 EXPECTED RESULTS

```
SUCESSO:
├─ Swipe direita = verde + like
├─ Swipe esquerda = vermelho + unlike
├─ Click = detalhe
├─ Sem console errors
└─ API chamadas corretas

FALHA:
├─ Swipe não funciona
├─ Cores não aparecem
```
